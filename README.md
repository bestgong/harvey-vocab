# Harvey 单词乐园 🦔📚

基于 **Harvey's Vocabulary Book** 的儿童英语单词复习与测试工具。纯静态网页（HTML / CSS / JS），可直接部署到 GitHub Pages。

## ✨ 功能

- **📚 单词本**：1554 个单词卡片浏览，支持按 *来源课本 / 词性 / 掌握状态* 筛选，关键词搜索，字母排序。点击卡片查看释义、例句、用法备注，可标记「已掌握」。
- **🎯 测试模式（四种）**
  - 中译英（看中文选英文，4 选 1）
  - 英译中（看英文选中文，4 选 1）
  - 例句填空（在句子空格里填入正确单词）
  - 单词拼写（看中文拼出英文，带首末字母提示）
- **📕 错题本**：答错自动收录并按错误次数排序，可作为专项题库再次练习。
- **进度保存**：掌握状态、错题记录保存在浏览器本地（localStorage），无需登录。
- **🌗 深色模式 + 移动端自适应**。

## 🔄 数据来源

应用启动时会**优先尝试读取在线 Google 表格**（实时最新单词）；若表格未公开或读取失败，则自动回退到打包在仓库中的离线数据 `data.js`。右上角的圆点会显示当前数据来源：

- 🟢 **在线实时** — 已成功连接公开表格
- 🟡 **本机数据** — 使用打包的离线数据

### 启用「在线实时」

1. 打开 Google 表格 → 右上角「共享」
2. 通用访问权限改为 **「任何知道链接的人」→ 查看者**
3. 刷新网页（或点右上角刷新按钮）即可读取最新单词

> 表格 ID：`14SHVvp1axGL9hThYTKc2Q78sL5C0pCNIad-16ZZsth0`

## 🔁 更新离线数据

当向表格里追加新课本后，重新生成 `data.js` / `data.json` 并推送即可（即使不开公开表格，离线数据也会更新）。

## 🚀 本地运行

```bash
# 任意静态服务器即可
python3 -m http.server 8000
# 打开 http://localhost:8000
```

## 📦 部署

已通过 GitHub Pages 发布。仓库 Settings → Pages → Source 选择 `main` 分支根目录即可。
